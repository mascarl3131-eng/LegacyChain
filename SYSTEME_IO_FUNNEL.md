## Recommended Systeme.io setup

1. Create a new funnel.
2. Choose “Build an audience”.
3. Add the landing page copy above.
4. Add a thank-you page with the app button.
5. Create a tag: `legacychain-lead`.
6. Create an email campaign: `The Chain Legacy Starter`.
7. Add the 4 emails above.
8. Connect the form to the tag and campaign.
9. Use the domain or subdomain:
   - `go.thechainlegacy.com` if possible
   - or keep a Systeme.io temporary URL at first
